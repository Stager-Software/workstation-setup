# Workstation setup

At Stager, we mostly run on Linux machines. We currently have two 'officially supported' distributions: Linux Mint and OpenSUSE Tumbleweed

OpenSUSE Tumbleweed used to be our sole recommended distribution, but it has since been replaced by Linux Mint due to it generally feeling more stable and way more user friendly for the Linux newbies among us. There are still a few people at Stager that use OpenSUSE Tumbleweed though, hence why this repository still contains all the necessary guides for it.

- To see more about installing a new Linux Mint machine, refer to [/linux/mint/setup.md](/linux/mint/setup.md)
- To see more about installing a new OpenSUSE Tumbleweed machine, refer to [/linux/opensuse-tumbleweed/setup.md](/linux/opensuse-tumbleweed/setup.md)
